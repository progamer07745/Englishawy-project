# 🚀 Englishawy Platform - React Expert Edition

A high-performance, responsive web platform built with **React.js** and **Tailwind CSS**, designed for English language learning services.

## 🛠️ Tech Stack
- **Frontend:** React.js (Hooks, Functional Components)
- **Styling:** Tailwind CSS (Utility-first, Custom Themes)
- **Animations:** Framer Motion (Smooth UI transitions)
- **Icons:** Lucide-React
- **Deployment:** Cloudflare

## ✨ Key Features & Technical Wins
- **Advanced Modal Logic:** Implemented a custom fixed-positioning system to ensure modals remain centered in the viewport during page scrolling, overcoming standard CSS stacking context limitations.
- **Dynamic Routing:** Integrated WhatsApp API for direct technical support communication.
- **Performance Optimized:** Zero-lag animations and optimized asset loading.
- **Responsive UI:** Fully compatible with all mobile devices and screen sizes.

## 🏗️ How to Run Locally
1. Clone the repo: `git clone https://github.com/progamer07745/Englishawy-project.git`
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
